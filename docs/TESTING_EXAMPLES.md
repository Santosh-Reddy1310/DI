## üí° Tips for Testing

- **Vary the weights**: Try different importance levels for criteria
- **Add constraints**: Test budget, timeline, and risk constraints
- **Test search**: Use the search bar to find decisions
- **Test filters**: Filter by status (draft, analyzing, complete)
- **Test CRUD**: Edit, duplicate, archive, and delete decisions
- **Test PDF export**: Generate reports for different decisions
- **Test What-If**: Adjust criteria weights and see ranking changes
- **Test mobile**: Try on different screen sizes
